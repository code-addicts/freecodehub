Stay Informed with React

Keep up-to-date with the latest from the React team. Read our blog for updates, release notes, and deprecation notices. Follow @reactjs on Twitter for quick highlights.

Detailed changelogs for every release are available in the CHANGELOG.md file in the React repository and on the Releases page.



Versioned Documentation

Always refer to the latest stable version of React documentation. For React 16 and later, find older versions on a separate page. Note that documentation for past versions is snapshotted at release and isn't continuously updated.



Something Missing?

If you find gaps in the documentation or something confusing, file an issue in the documentation repository with your suggestions for improvement or tweet at @reactjs. We value your feedback!
